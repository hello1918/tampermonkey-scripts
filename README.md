## access-link 跳转链接修复

### 🔧 插件功能

- [x] 手动点击跳转、3 秒后自动跳转
- [x] 微信拦截页面跳转
- [x] 知乎拦截页面跳转

### 💽 安装插件

[access-link 插件安装地址](https://greasyfork.org/zh-CN/scripts/395970)
