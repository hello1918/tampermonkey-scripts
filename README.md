# access-link

油猴插件：在拦截页面增加跳转按钮，暂只支持微信和知乎

## 🔧 插件功能

- [x] 手动点击跳转、3 秒后自动跳转
- [x] 微信拦截页面跳转
- [x] 知乎拦截页面跳转

## 💽 安装插件

请确保已先安装 [Tampermonkey](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)

[access-link 插件安装地址](https://greasyfork.org/zh-CN/scripts/395970)
